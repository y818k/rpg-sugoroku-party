# RPG Sugoroku Party

ブラウザで遊べる、最大4人対戦のRPG風すごろくゲームMVPです。Next.js + TypeScript + Socket.IO で、1台PC内の複数ブラウザ/タブからルーム参加して遊べます。

## 起動方法

```bash
npm install
npm run dev
```

起動後、`http://localhost:3000` を開きます。別タブや別ブラウザで同じURLを開き、ルームコードを入力すると最大4人まで参加できます。

## 実装済み

- ルーム作成、ルームコード参加、最大4人ロビー
- ホストによるゲーム開始
- Socket.IO によるリアルタイム状態同期
- ステージ1/2/3、中ボス、村、魔王を含むランダムマップ
- ターン制、1〜6ルーレット、アイテムによる出目補正
- 戦闘、宝箱、イベント、村、ボス、魔王の処理
- 冒険者、戦士、魔法使い
- HP/MP/物理攻撃/魔法攻撃/防御/レベル/経験値/ゴールド/スコア
- 武器、防具、アクセサリー、アイテム所持と装備変更
- 村での回復、ショップ、売却、初回転職
- 30種類のイベントと簡易重複制御
- 切断中表示と同じプレイヤーIDでの再接続
- 魔王撃破によるゲーム終了と最終スコア画面
- スマホ幅向けレスポンシブUI

## MVP上の割り切り

- ルーム状態はサーバープロセスのメモリ上に保持します。サーバー再起動で消えます。
- 外部DB、ログイン、課金、広告、チャットは未実装です。
- 3ルート分岐はマスにルートラベルを持たせた簡易表現です。実際の分岐選択UIは次フェーズ想定です。
- 切断プレイヤーのAI操作は未実装で、MVPでは切断中表示と再接続に対応しています。
